Bienvenidos a nuestro proyecto Usando SQlite + Phyton.

El trabajo realizado busca hacer una simulación del sistema que se aplica en nuestra escuela, de este modo para realizarlo se usaron dos programas que permiten facilitar el uso de una base de datos: “python” que permite crear nuestro código y “SQlite administrator” que permite crear la base de datos.
